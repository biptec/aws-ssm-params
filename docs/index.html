aws-ssm-params

A CLI/TUI for AWS SSM Parameter Store discovery, filtering, import, export, and editing.

Commands

aws-ssm-params [global options] <command>

interactive
get
put
import
export

Global options

--region value       repeat for multiple regions
--all-regions
--profile value
--no-color
--keymap value

CLI lists use repeated flags. Comma-separated lists are supported only in environment variables such as AWS_SSM_PARAMS_REGION.

Filters

Filters use field:extglob. A bare filter is a name filter. Conditions inside one filter are joined by AND with ;. Multiple filters are joined by OR.

--filter 'name:/prod/*;region:eu*'
--filter '/app/prod/**'

--filters-file uses the same syntax with one OR group per line. Blank lines and # comments are ignored.

JSON fields

--field aws_field[:file_field] maps AWS fields to import/export file fields. JSON uses arrays by default. --json-key-field name switches JSON to an object keyed by the selected AWS field.
